## Quick non-coder deploy (Vercel)

1. Create a GitHub repo named `rent-tracker-frontend` and upload these files (extract the zip).
2. In Vercel, import the project from GitHub.
3. Add Environment Variable: REACT_APP_API_URL = https://your-backend-url (from Render)
4. Deploy. Open the Vercel URL on your mobile.
